What this app does:

This app allows you to interact with various elements on the screen using different jQuery effects.

  • Slide toggle: You can slide the rainbow image in and out using the "Slide Toggle" button.
  • Fade toggle: The butterfly image can fade in and out using the "Fade Toggle" button.
  • Change styles: Change the background colour and font style of the text in the middle section.
  • Slide effects: The spooky house image slides out and slides back in using the "Slide House Out" and "Slide House In" buttons.
  • Hide/Show effects: Hide and show the bulb image using the respective buttons.
  • Fade effects: Fade the ghost image in and out using the "Fade Ghost Out" and "Fade Ghost In" buttons.
  • Toggle header: Toggle the visibility of the entire header using the new "Toggle Header Visibility" button.
  • Change text: Change the description of the ghost with the "Change Ghost Description" button.
